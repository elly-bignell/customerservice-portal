No-Index Tag Not Set

0/5
Complete manual testing on both desktop and mobile
Contact Forms Working
A broken contact form means the site is actively losing leads without the client knowing. Test the form end to end and confirm submission emails are received.
Complete and submit the main contact form using a test email address. Confirm the submission email is received and a thank-you message is shown.
Pass: Form submits successfully and confirmation is received
Critical
Phone Number Links Working
On mobile, phone numbers should be tappable links that trigger a call. Non-linked phone numbers are a significant source of lost leads on mobile devices.
On a mobile device, tap the phone number on the site. It should prompt a call.
Pass: Phone numbers are clickable on mobile and trigger a call
Important
Mobile Display
Check the overall mobile layout is functional and presentable. Look for overlapping text, broken sections or content that extends off screen.
View the site on a mobile device or use Chrome DevTools responsive mode. Check homepage, service pages and contact page.
Pass: Site displays correctly on mobile with no layout issues
Important
Navigation Links Working
Broken navigation links frustrate visitors and waste the crawl budget Google allocates to the site.
Click through all header navigation links and confirm each one loads the correct page with no 404 errors.
Pass: All navigation links load without errors
Important
Image Alt Text Spot Check
Alt text describes images to search engines and screen readers. Missing alt text is a missed opportunity for keyword relevance and an accessibility issue that can affect rankings.
Right-click a main image > Inspect > check for an alt attribute. Or run SE Rankings Site Audit and look for missing image alt text.
Pass: Key images have relevant, descriptive alt text
Recommended

A site that looks significantly dated can undermine trust with visitors even if the content is strong, directly reducing conversion rates. This is a subjective but important observation to flag at onboarding — if a redesign is warranted, the earlier it is raised with the client the better.
Review the site as a first-time visitor would. Consider: does the design look current? Does it reflect the quality of the client's business? Would a prospect trust it?
